# unrealEnginePractice
Following a tutorial to learn Unreal Engine 7.7

## Clone And Open On Another Machine

This project uses Git LFS for Unreal binary assets. Install Git LFS before cloning so assets are downloaded correctly.

1. Install Git LFS.
2. Run `git lfs install` once on your machine.
3. Clone the repository.
4. Open `unrealEnginePractice/unrealEnginePractice.uproject` in Unreal Engine.

If Git LFS is not installed, Unreal asset files will appear as missing because only pointer files are downloaded.
